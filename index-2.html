CAREERS

Front-end Development

Front-end web development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly.




Back-end Development

A back-end web developer is responsible for server-side web application logic and integration of the work front-end developers do. Back-end developers usually write the web services and APIs used by front-end developers and mobile application developers.

Design

A designer, web or graphic, works on the look and feel of a website or software. Web design encompasses several different aspects, including webpage layout, content production, and graphic design.
