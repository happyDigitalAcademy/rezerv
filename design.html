Design

There are various tools essential for learning web/graphic design. Make your way through the following
starting with photoshop. After this you will be able to decide which tool you work best with.

Photoshop

  • A multi-purpose application for creating (pixel-based) artwork suitable for both screen and print graphics.
+

Illustrator

  • A vector based program fro creating detailed and scalable (vector-based) art for almost any use.
+

InDesign

  • InDesign seamlessly integrates photo manipulations and vector art created in Photoshop and Illustrator.
+

Dreamweaver

  • A web design and development application that combines a visual design surface known as Live View and a code editor.
+

Sketch

  • A vector design tool entirely focused on user interface design.
+

HTML/CSS

  • HTML is used for creating web pages and web applications, which are styled using CSS.
+
